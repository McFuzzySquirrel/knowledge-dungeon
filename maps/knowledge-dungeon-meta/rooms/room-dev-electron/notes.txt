# Electron Packaging

## Overview

Electron bundles the web app as a native desktop application for macOS, Windows, and Linux.

## The Electron Architecture

### Main Process
Runs in Node.js:
- Creates windows
- Handles file system operations
- Manages app lifecycle
- Responds to menu commands

### Renderer Process
Runs the web app:
- React components
- Phaser game
- Zustand stores
- Event listeners

### Preload Bridge
Secure communication:
- Exposes safe APIs to web code
- Example: `window.electronAPI.loadSubjects()`
- Prevents XSS attacks

## The Build Process

```
1. npm run build:web     # Build React + Phaser bundle
   ↓
   dist/ (optimized web assets)

2. npm run build:electron  # Bundle Electron main process
   ↓
   dist-electron/ (Electron app)

3. npm run package:electron:mac  # electron-builder packages it
   ↓
   release/ (installer DMG, ZIP)
```

## Packaging Targets

### macOS
```bash
npm run package:electron:mac
```

Produces:
- `.dmg` — Disk image (double-click to install)
- `.zip` — For direct download

**Code signing** (optional):
- Requires Apple Developer certificate
- Prevents "unidentified developer" warning
- Set `CSC_IDENTITY_AUTO_DISCOVERY=false` to skip

### Windows
```bash
npm run package:electron:win
```

Produces:
- NSIS installer (`.exe`)
- `.zip` — Portable executable

**Code signing** (optional):
- Requires Authenticode certificate
- Set `CSC_LINK` and `CSC_KEY_PASSWORD` env vars

### Linux
```bash
npm run package:electron:linux
```

Produces:
- `.AppImage` — Single executable, works anywhere
- `.deb` — Debian package (install via apt)
- `.AppImage.zsync` — Auto-update metadata

## Configuration

### electron-builder.config.js

```javascript
export default {
  appId: 'com.mcfuzzysquirrel.knowledge-dungeon',
  productName: 'Knowledge Dungeon',
  files: ['dist/**/*', 'dist-electron/**/*'],
  mac: {
    target: ['dmg', 'zip'],
    icon: 'build/icon.icns'
  },
  win: {
    target: ['nsis', 'zip'],
    icon: 'build/icon.ico'
  },
  linux: {
    target: ['AppImage', 'deb'],
    icon: 'build/icon.png'
  }
};
```

## App Updates

Currently: **Manual downloads** from GitHub releases.

Potential future: **Auto-update** via:
- `electron-updater` (download from GitHub releases)
- Custom server (serve updates from your own server)
- Code signing required for trusted updates

## Troubleshooting

### "Cannot find module" error
- Ensure `npm install` was run
- Check `dist-electron/` was built

### DMG fails to create (macOS)
- Install `dmg` package: `brew install dmg`
- Or skip with `--mac zip` only

### NSIS installer fails (Windows)
- Install `nsis` package: `choco install nsis` or `scoop install nsis`
- Or skip with `--win portable` only

## Design Insight

Electron makes Knowledge Dungeon feel like a **professional native app** while keeping it a single web codebase. The main/preload separation ensures web code stays sandboxed and secure. electron-builder handles the complexities of cross-platform packaging, freeing us to focus on features.
