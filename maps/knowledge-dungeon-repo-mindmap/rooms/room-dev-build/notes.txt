# Build Scripts & Commands

## Package.json Scripts

All build-related scripts are in `package.json`. Here's what each does:

### Web Development
```bash
npm run dev                    # Vite dev server (localhost:5173)
npm run build:web            # Production web bundle
npm run preview:web          # Preview built bundle locally
```

### Electron Development
```bash
npm run electron             # Web build + Electron dev mode
npm run build:electron       # Build web + Electron main process
```

### Desktop Packaging
```bash
npm run package:electron         # Current OS (unpacked, fastest for testing)
npm run package:electron:mac     # macOS DMG + ZIP
npm run package:electron:win     # Windows NSIS + ZIP
npm run package:electron:linux   # Linux AppImage + deb
npm run package:electron:full    # All three platforms
```

### Testing & Quality
```bash
npm run test                 # Run tests once
npm run test:watch          # Watch mode
npm run lint                # Check style
npm run lint --fix          # Fix auto-fixable issues
npm run typecheck           # Check TypeScript types
npm run check:bundle-size   # Verify bundle size
```

## Build Configuration

### Vite (`vite.config.ts`)
- Dev server settings
- Build output configuration
- Asset handling
- Plugin configuration

### TypeScript (`tsconfig.json`)
- Compilation target (ES 2020)
- Module system (ES modules)
- Strict mode enabled
- Path aliases for imports

### Electron Builder (`electron-builder.config.js`)
- Application metadata
- Icon and installer settings
- Platform-specific configurations
- Signing and notarization options

### ESLint (`eslint.config.js`)
- Linting rules
- TypeScript plugin configuration
- Rule overrides by file type

## Understanding the Build Pipeline

```
source code (TypeScript, JSX, CSS)
        ↓
   [Vite transpiles]
        ↓
 optimized JavaScript, CSS, assets
        ↓
    [tree-shake, minify]
        ↓
  dist/ production bundle
        ↓
 [Electron builder bundles]
        ↓
 release/ native installers
```

## Build Performance

- **Dev server**: Starts in <500ms
- **Hot reload**: Module updates in <1s
- **Full build**: ~30 seconds
- **Electron package**: ~60 seconds (platform-dependent)

These timings keep the workflow snappy.

## Design Insight

All scripts are configured to be **self-documenting**. Running `npm run` lists all available commands with descriptions. The build pipeline is **modular**—you can build web-only without Electron, or test without building, keeping feedback cycles tight.
