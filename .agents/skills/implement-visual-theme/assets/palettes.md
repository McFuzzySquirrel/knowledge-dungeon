# Palette Definitions

Full palette examples for the Knowledge Dungeon visual theme. See `docs/research/ui-enhancements.txt` for the original research.

## Dark Fantasy

```
--bg-dark:       #1a1025  (deep purple-black, backgrounds)
--bg-panel:      #2a1a3e  (dark purple, panels)
--accent-gold:   #c9a84c  (muted gold, highlights)
--text-primary:  #e8dcc8  (warm off-white, text)
--accent-blue:   #4a7fb5  (muted blue, interactive elements)
```

### Hex-to-0x Conversion (Dark Fantasy)

| CSS Variable | Hex (CSS) | 0x (Phaser) |
|-------------|-----------|-------------|
| `--bg-dark` | `#1a1025` | `0x1a1025` |
| `--bg-panel` | `#2a1a3e` | `0x2a1a3e` |
| `--accent-gold` | `#c9a84c` | `0xc9a84c` |
| `--text-primary` | `#e8dcc8` | `#e8dcc8` (Phaser text uses hex strings) |
| `--accent-blue` | `#4a7fb5` | `0x4a7fb5` |
